## Usage

Install the module globally, and access it on the command line:

```bash
$ nsp --help

Usage: cli [options] <directory>

Options:

  -V, --version                 output the version number
  -n, --name <string>           Name of the new project
  -d, --desc <string>           Project description
  -an, --author-name <string>   Name of the project author
  -ae, --author-email <string>  Email of the project author
  -u, --username <string>       GitHub username
  -h, --help                    output usage information
```

`<directory>` is the only required cli argument. If you don't enter any of the other options, you will be prompted to provide an answer.
